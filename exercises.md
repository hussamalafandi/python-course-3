# **Exercise Sets by Unit**

---

## **Unit 1 – Introduction to Programming & Python Ecosystem**

### Core Exercises

1. Install Python and verify the version from the terminal.
2. Create and activate a virtual environment.
3. Install `numpy` and `pandas` inside the environment.
4. Create:

   * One Python script (`hello.py`)
   * One Jupyter Notebook (`intro.ipynb`)
5. Run the same code in both environments.

### Stretch Exercises

* Compare notebooks vs. scripts: write a short reflection.
* Install an additional package and document the steps in Markdown.

---

## **Unit 2 – Python Fundamentals: Syntax, Variables & Types**

### Core Exercises

1. Write a program that:

   * Asks for a name and age
   * Prints a formatted sentence using both values
2. Convert user input from string to integer safely.
3. Write expressions using all operator types.
4. Identify and fix at least three syntax or type errors.

### Stretch Exercises

* Write a temperature converter (C ↔ F).
* Implement basic input validation with error messages.

---

## **Unit 3 – Control Flow & Program Logic**

### Core Exercises

1. Write a program that:

   * Determines if a number is positive, negative, or zero
2. Create a loop that prints all even numbers from 1–100.
3. Implement a simple menu system using `while`.
4. Use `break` and `continue` meaningfully.

### Stretch Exercises

* Number guessing game with limited attempts.
* Prime number checker.

---

## **Unit 4 – Core Data Structures**

### Core Exercises

1. Store and manipulate a list of student grades.
2. Convert a list to a set and explain the result.
3. Build a dictionary-based phone book.
4. Iterate over dictionary keys and values.

### Stretch Exercises

* Word frequency counter for a text input.
* Inventory system with add/remove functionality.

---

## **Unit 5 – Functions, Modularity & Code Organization**

### Core Exercises

1. Rewrite earlier scripts using functions.
2. Write functions that:

   * Accept parameters
   * Return values
3. Demonstrate local vs. global scope.
4. Create a simple utility module and import it.

### Stretch Exercises

* Recursive vs. iterative implementation of a function.
* Create a math utilities module with documentation.

---

## **Unit 6 – Mini Project I: Programming Foundations**

### Required Project

Choose **one**:

* Budget tracker
* Quiz application
* Text analysis tool

### Requirements

* Minimum of 5 functions
* At least one data structure
* Clear user interaction
* Clean, readable code

### Stretch

* Save/load data from a file.
* Add basic error handling.

---

## **Unit 7 – Object-Oriented Programming (OOP): Foundations**

### Core Exercises

1. Define a class with attributes and methods.
2. Use a constructor to initialize state.
3. Create multiple instances.
4. Add a method that modifies object state.

### Stretch Exercises

* Implement basic inheritance.
* Compare procedural vs. OOP versions of the same program.

---

## **Unit 8 – Advanced OOP & Design Principles**

### Core Exercises

1. Add encapsulation using private attributes.
2. Implement an abstract base class.
3. Use class and static methods appropriately.
4. Apply composition instead of inheritance.

### Stretch Exercises

* Implement the Factory Pattern.
* Refactor an earlier OOP solution using better design.

---

## **Unit 9 – Computational Thinking & Algorithms**

### Core Exercises

1. Implement linear and binary search.
2. Implement bubble sort.
3. Measure execution time of algorithms.
4. Write a recursive function.

### Stretch Exercises

* Implement quick sort.
* Visualize algorithm performance.
* Solve a classic recursion problem (e.g., Towers of Hanoi).

---

## **Unit 10 – Numerical Computing with NumPy**

### Core Exercises

1. Create NumPy arrays from lists.
2. Perform element-wise operations.
3. Replace loops with vectorized operations.
4. Compute basic statistics (mean, std).

### Stretch Exercises

* Simulate random data.
* Compare NumPy vs. pure Python performance.

## **Unit 11 – Data Analysis with Pandas**

### Core Exercises

1. Load data from CSV and Excel.
2. Inspect data using `head`, `info`, `describe`.
3. Handle missing values.
4. Filter and group data.
5. Compute aggregated statistics.

### Stretch Exercises

* Merge multiple datasets.
* Perform time-based grouping.

---

## **Unit 12 – Data Visualization & Reporting**

### Core Exercises

1. Create:

   * Line plot
   * Bar chart
   * Scatter plot
2. Label axes and titles correctly.
3. Choose appropriate charts for given data.

### Stretch Exercises

* Create multi-plot dashboards.
* Improve plot aesthetics for presentations.
* Write a short analytical summary.

## **Unit 13 – Version Control, Prototyping & Agile Methods**

### Core Exercises

1. Initialize a Git repository.
2. Commit changes with meaningful messages.
3. Create and merge a branch.
4. Push to GitHub.
5. Write a short README.

### Core Exercises

2. Run tests and interpret failures.
